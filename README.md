# Princípios Matemáticos do Machine Learning - Implementações Práticas

Este repositório contém uma série de scripts em Python desenvolvidos como parte de um projeto de Iniciação Científica focado nos fundamentos matemáticos do Aprendizado de Máquina. O projeto tem como base o livro *"Mathematics for Machine Learning"* (Deisenroth, Faisal, Ong).

O objetivo principal destes códigos é **traduzir conceitos matemáticos** (Álgebra Linear, Probabilidade e Otimização) em **visualizações computacionais práticas e interativas**.

---

## Tecnologias Utilizadas
* Python 3.x
* [NumPy](https://numpy.org/) - Operações matriciais e álgebra linear avançada
* [Matplotlib](https://matplotlib.org/) - Plotagem e visualização de dados
* [Scikit-Image](https://scikit-image.org/) - Processamento de imagens de teste

Para instalar as dependências necessárias, execute:
```bash
pip install numpy matplotlib scikit-image
